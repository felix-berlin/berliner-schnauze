Kleines Release mit einem sichtbaren Effekt außerhalb der Seite: Geteilte Links sehen jetzt ordentlich aus.

---

## Link-Vorschauen beim Teilen

Wer einen Link zur Seite in WhatsApp, Mastodon oder sonstwo teilt, bekommt jetzt automatisch ein passendes Vorschaubild dazu. Für jede Wortseite wird ein eigenes Bild generiert, mit dem Wort drauf. Vorher gab es an vielen Stellen nur nackten Text, was beim Teilen ziemlich trist aussah.

---

## Verbesserungen im Hintergrund

- Adressen der geteilten Links auf die aktuelle Domain umgestellt
- Interne Korrekturen daran, wie Seiten sich Suchmaschinen gegenüber ausweisen
