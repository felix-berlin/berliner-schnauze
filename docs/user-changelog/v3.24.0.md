---

## Wortarten bequemer auswählen

Der Filter nach Wortarten (Nomen, Verb, Adjektiv und so weiter) bestand bisher aus einer Reihe einzelner Buttons, die bei vielen Wortarten schnell unübersichtlich wurde. Er ist jetzt ein Dropdown-Menü: antippen, gewünschte Wortarten anhaken, fertig. Das Menü bleibt beim Auswählen offen, sodass sich mehrere Wortarten hintereinander markieren lassen, ohne es jedes Mal neu öffnen zu müssen. Ausgewählte Wortarten lassen sich direkt im Dropdown auch wieder abwählen.
