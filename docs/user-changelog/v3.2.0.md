## Fehlerseite im neuen Design

Seit dem Redesign fehlte eine eigene Seite für kaputte Links oder falsche Adressen. Die gibt es jetzt wieder, passend zum neuen Look: Wer eine nicht existierende Adresse aufruft, landet auf einer klar gestalteten Hinweisseite statt im Nichts.
