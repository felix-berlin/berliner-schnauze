## Bug melden direkt aus dem Menü

Im Header-Menü ist ein neuer Link dazugekommen, der direkt zur Fehlermeldung auf GitHub führt. Wer also über einen Bug stolpert, muss nicht erst lange suchen, wo und wie er sich melden lässt.
