---

## Installierbar wie eine App

Wer die Seite besucht, bekommt jetzt die Möglichkeit, sie auf dem Smartphone oder Rechner zu installieren, mit eigenem Icon auf dem Homescreen statt nur als Lesezeichen im Browser.

- Eigenes Icon fürs Homescreen, in mehreren Größen für unterschiedliche Geräte
- Kurzbeschreibung ergänzt, die beim Installieren angezeigt wird
- Passende Farben und Vorschaubilder für den Installationsdialog
