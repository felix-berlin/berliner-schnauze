---

## Navigation kürzt sich selbst

Die kleine Pfadanzeige oben auf jeder Wortseite (Start, Wörter, und so weiter) zeigt bei langen Pfaden nicht mehr jede einzelne Station, sondern kürzt die mittleren Teile mit drei Punkten ab. Auf schmalen Bildschirmen bleibt der Pfad dadurch lesbar, statt umzubrechen oder abgeschnitten zu werden.
