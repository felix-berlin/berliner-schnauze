Direkt im Anschluss an die App-Installation vom letzten Update noch eine Ergänzung, die schon länger fehlte.

---

## Alle Wörter auf einen Blick

Es gibt jetzt eine eigene Übersichtsseite, auf der sämtliche Wörter im Wörterbuch gelistet sind, mit eigenem Layout statt nur über die Suche erreichbar.

## Schneller zurück auf der Wortseite

Auf einzelnen Wortseiten führt ein neuer Zurück-Button direkt zur vorherigen Ansicht, ohne über die Suche oder das Menü umzuwegen.

---

## Verbesserungen im Hintergrund

- Darstellung von Wortgruppen korrigiert: Die Liste stand vorher nicht sauber in einer Linie mit der Überschrift der jeweiligen Gruppe
