Das war ein größeres Update mit einem komplett neuen Bereich der Seite. Es gibt jetzt ein Magazin mit Artikeln rund um den Berliner Dialekt, dazu ein paar Verbesserungen, die direkt daraus entstanden sind.

---

## Neu: das Magazin

Unter berliner-schnauze.wtf/magazin sammeln sich ab sofort Artikel zu Geschichte, Herkunft und Kuriositäten der Berliner Schnauze, von Redewendungen bis zu einzelnen Berolinismen. Die Übersichtsseite zeigt alle Artikel mit Titelbild, die Einzelseiten holen ihre Inhalte direkt aus der Redaktion.

- Artikeltexte werden Block für Block aufgebaut, Absätze, Bilder und Zitate bekommen jeweils ihr eigenes, zur Seite passendes Aussehen statt eines generischen Einheitslooks
- Jeder Artikel zeigt oben eine geschätzte Lesezeit in Minuten
- Titelbilder werden in modernen Bildformaten ausgeliefert, damit sie schnell laden

---

## Inhaltsverzeichnis für lange Artikel

Hat ein Artikel mindestens drei Zwischenüberschriften, erscheint daneben automatisch ein kleines Inhaltsverzeichnis. Beim Scrollen markiert es von selbst den Abschnitt, der gerade gelesen wird, sodass man in längeren Texten nie den Überblick verliert.

---

## Amazon-Partner-Hinweis

Verlinkt ein Artikel auf Amazon, taucht direkt am Anfang ein kurzer Hinweis auf: Ich verdiene an solchen Verkäufen eine kleine Provision, der Preis ändert sich dadurch für niemanden. Die betroffenen Links sind außerdem einzeln gekennzeichnet.

---

## Navigation aus einer Hand

Das Hauptmenü der Seite wird jetzt zentral aus der Redaktion gepflegt statt fest im Code zu stehen. Neue Einträge, etwa der Link zum neuen Magazin, lassen sich dadurch einpflegen, ohne dass für jede Kleinigkeit eine neue Version der Seite gebaut werden muss.

---

## Verbesserungen im Hintergrund

- Interne Links innerhalb von Artikeln funktionieren jetzt als normale interne Verlinkung, nicht mehr als Sprung auf eine absolute Adresse
- Kleinere optische Korrekturen am Inhaltsverzeichnis und an den Buttons am Ende eines Artikels
- Zusätzliche Absicherungen beim Verarbeiten von Redaktionsinhalten, damit keine fehlerhaften Inhalte durchrutschen
