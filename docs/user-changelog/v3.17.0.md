Größeres Update unter der Haube: Die Wortsuche wurde technisch von Grund auf neu gebaut und läuft jetzt spürbar runder.

---

## Neues Suchfenster

Die Suche läuft jetzt komplett im Browser statt über einen externen Dienst, dadurch reagiert sie schneller auf Eingaben. Das Suchfenster selbst hat ein neues Layout bekommen:

- Eine Anzeige zeigt direkt, wie viele Treffer eine Suche liefert
- Wer nichts findet, sieht statt einer leeren Fläche eine Meldung im Berliner Ton: "Da biste anjemeiat. Keen Treffer."
- Durch die Ergebnisliste lässt sich jetzt auch komplett mit der Tastatur navigieren, inklusive eigener Tastenkürzel zum Öffnen, Schließen und Springen zwischen Treffern
- Auf der Startseite, wo bereits ein eigenes Suchfeld sitzt, ist das zusätzliche Such-Icon im Header verschwunden, das war doppelt gemoppelt

---

## Verbesserungen im Hintergrund

- Ähnliche Wörter auf der Wortseite sind jetzt besser mit Screenreadern nutzbar
- Suchindex und Wort des Tages werden zwischengespeichert, dadurch laden wiederholte Suchen schneller und funktionieren teilweise auch offline
