Kleines, aber praktisches Update: Wer jetzt auf einen toten Link klickt oder eine falsche Adresse eintippt, landet nicht mehr im Nichts.

---

## Fehlerseite

Für nicht existierende Seiten gibt es jetzt eine eigene Fehlerseite statt einer leeren oder kryptischen Meldung. Das macht sofort klar, dass die aufgerufene Adresse nicht stimmt, statt einfach im Leeren zu verlaufen.
