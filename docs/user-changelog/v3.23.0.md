---

## Vorschlagsformular merkt sich den Suchbegriff

Wer nach einem Wort sucht, das (noch) nicht im Wörterbuch steht, bekommt den Hinweis, es vorzuschlagen. Bisher musste der gesuchte Begriff im Formular nochmal von Hand eingetippt werden. Jetzt wird er automatisch ins Vorschlagsformular übernommen, ein Klick genügt, um direkt weiterzumachen.
