## Filter und Suche bleiben erhalten

Sowohl der Buchstaben-Filter als auch die eingegebene Wortsuche werden jetzt lokal gespeichert. Nach einem Neuladen der Seite oder einem erneuten Besuch stehen beide wieder genau so da, wie sie zuletzt eingestellt waren.
