---

## Hinweis bei veralteten Browsern

Manche Browser sind so alt, dass moderne Web-Technik damit nicht mehr richtig läuft, Layout kaputt, Funktionen fehlen, Fehler an Stellen, wo eigentlich keine sein sollten. Ab sofort prüft die App beim Laden, ob der verwendete Browser noch unterstützt wird. Ist das nicht der Fall, erscheint eine Meldung mit der Empfehlung, den Browser zu aktualisieren. Der Hinweis lässt sich wegklicken, bleibt aber so lange sichtbar, bis das passiert.
