## Verbesserungen im Hintergrund

- Seiten werden vorgeladen, sobald die Maus über einem Link ist. Der Klick danach fühlt sich sofort an
- Seitenaufbau stabilisiert: weniger Layout-Sprünge, wenn Schriften nachladen
- Schriften laden cleverer, das verkürzt die Wartezeit beim ersten Besuch
- Ein Fehler beim Aufruf von Wörtern ohne eigene Adresse ist behoben, die App startet wieder sauber durch
