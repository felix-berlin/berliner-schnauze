## Verbesserungen im Hintergrund

- Seiten werden beim Hover vorgeladen — Links reagieren deutlich schneller

- Schriften laden jetzt cleverer — kürzere Wartezeit beim ersten Besuch
- Routing-Fehler bei Wörtern ohne URL behoben — die App startet jetzt sauber durch
