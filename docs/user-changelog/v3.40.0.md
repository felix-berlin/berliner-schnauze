Das Wörterbuch hat eine neue Ebene bekommen: Themen. Ich wollte schon lange, dass man Wörter nicht nur alphabetisch, sondern auch inhaltlich durchstöbern kann.

---

## Wörter nach Themen entdecken

Jedes Wort kann jetzt einem oder mehreren Themen zugeordnet sein. Das zeigt sich an drei Stellen:

- Auf der Wortseite stehen die Themen als kleine Badges direkt beim Wort
- Jedes Thema hat eine eigene Übersichtsseite mit allen zugehörigen Wörtern
- In der Suche gibt es einen neuen Themen-Filter. Mehrere Themen gleichzeitig auswählen geht auch

---

## Verbesserungen im Hintergrund

- Die Anzeige der Filter-Auswahl spricht jetzt durchgehend Deutsch
- Stabileres Laden der Themen, wenn der Server mal zickt
- Themenseiten sind für Suchmaschinen aufbereitet
