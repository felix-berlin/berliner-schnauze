## Browser-Farbe folgt dem Design

Je nachdem, ob die App gerade im hellen oder dunklen Modus angezeigt wird, färbt sich jetzt auch die Adressleiste beziehungsweise die Statusleiste des Browsers passend ein, etwa auf dem Smartphone. Das sorgt für einen einheitlicheren Eindruck, ohne dass mitten im dunklen Modus plötzlich ein heller Rand um die Seite herum auftaucht.
