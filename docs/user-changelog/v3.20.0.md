Ein kurzes Update mit einer kleinen, aber spürbaren Layout-Korrektur auf der Wortseite.

---

## Feinschliff für die Wortseite

Die Liste der Übersetzungen richtet sich jetzt nach der Breite des verfügbaren Platzes statt starr in einer festen Größe zu erscheinen. Bei Wörtern mit vielen Übersetzungen sieht das Layout dadurch aufgeräumter aus, egal auf welchem Bildschirm die Seite geöffnet wird.
