## Social-Media-Wechsel

Der Link zu Twitter ist aus Footer und Kopfbereich verschwunden, an seiner Stelle gibt es jetzt einen Link zum neuen Mastodon-Kanal. Außerdem wurde der Social-Media-Bereich im Header komplett entfernt, dort war er ohnehin eher ein Fremdkörper. Wer den Kanälen folgen möchte, findet den Link weiterhin im Footer der Seite.
