# Week 3 (W23): Jun 1–4, 2026 — Chasing the Numbers

A short, dense week: CLS hunted across six Lighthouse sessions on the first day, a German dictionary swapped out, word curiosities redesigned, and the word page broken into discrete section components — all in under four days of real calendar time.

---

## Narrative

Week 3 opens where Week 2 left off: Cumulative Layout Shift is still not fixed, Lighthouse is still misbehaving, and a branch named `perf-img-and-others` is still waiting on proof. Sessions 243 through 252 are one long CLS hunt. Desktop shows 0.693. Then 0.813. Mobile shows 0.00. The trace is read, the wrong containers are blamed, and two fixes land: an unnecessary `min-height: 280px` stripped from the `.c-facts-grid__wotd-slot` and a placeholder `min-height` removed from the mobile image path. By obs 1033, the performance trace reports LCP at 163ms and CLS at 0.00. The desktop regression is closed.

Font metrics took a hit along the way. Fallback font declarations introduced a CLS regression of their own — removed in obs 1018–1020. The `isIndex` detection in `MainHeader` was also broken for the `index.html` pathname; fixed in obs 1016.

Jun 1 ends quietly with the PWA. `onNeedReload` is confirmed available in vite-plugin-pwa 1.3.0, and a native notification on PWA update is implemented (obs 1049). The session that prompted this is listed as session 254 — a sharp pivot from performance work.

Jun 3 brings the word page. The feature prompt is to expand "Wortkuriositäten" (word curiosities). Before touching content, the sidebar sticky positioning gets a round of fixes — four observations, one per breakpoint — culminating in a dynamic CSS variable synchronized to actual header height (obs 1055). Then the curiosities function gains eight new linguistic properties, a stats dashboard UI lands, a dedicated SCSS file is created, and the test suite expands to cover the new cases (obs 1058–1062).

Jun 4 begins around midnight and runs into the early hours. Felix asks why Claude rarely uses Claude-Mem, Graphify, and Serena — a meta-session. The answer is a broken `UserPromptSubmit` hook with an unclosed quote. A shell script replaces the inline JSON, permissions are updated, and the tool checklist is restored (obs 1066–1070). Then a `PostToolUse` hook for auto-running tests on save is configured. Letter frequency bars get a dynamic OKLCH gradient instead of repeating colors. The monolithic word page is refactored: section components extracted (obs 1080). Accessibility and memory leak analysis follows. Then, in a sequence of five sessions (265–268), the question of German noun gender surfaces: `de-compromise` has no gender API, `find-artikel` is investigated, and the Wiktionary API is refactored to extract grammatical gender reliably (obs 1094). The week ends with a native dependency issue in `claude-mem` — tree-sitter binaries missing — reported but not resolved within the week's observations.
